```
- Java 21 (>= java 8+)
- Junit 5
- AssertJ

```

# Getting started

## Prerequisites
- Any IDE of your choice and preference which supports Java
- Java JDK 8+ 
- Maven 3.x

## How to run
```
$ git clone https://github.com/salgotraja/dsa.git
$ cd dsa
$ mvn clean package
$ mvn test
```
